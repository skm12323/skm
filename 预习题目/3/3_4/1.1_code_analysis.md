# §1.1 代码分析

```python
from matplotlib import pyplot as plt
import numpy as np
import tensorcircuit as tc
```
- 导入画图（matplotlib）、数值计算（numpy）、量子线路库（tensorcircuit）。

```python
K = tc.set_backend("tensorflow")
```
- 选定计算后端为 TensorFlow。`K` 是后端对象，后面所有张量运算、求导、编译都通过它调用。

```python
@K.jit
def exp_z(param):
    c = tc.Circuit(1)
    c.rx(0, theta=param)
    return K.real(c.expectation_ps(z=[0]))
```
- `@K.jit`：把这个函数 JIT 编译，反复调用时加速。
- `tc.Circuit(1)`：新建一个 **1 比特**量子线路，初态默认为 |0⟩。
- `c.rx(0, theta=param)`：在第 0 号比特上加一个 **RX(param)** 门，即绕 X 轴旋转 param 弧度：$R_x(\text{param})=e^{-i\,\text{param}\,X/2}$。此时态变为 $R_x(\theta)|0\rangle$。
- `c.expectation_ps(z=[0])`：测量第 0 号比特的 **Z 算符期望** $\langle Z\rangle$。
- `K.real(...)`：取实部（期望理论上是实数，取实部确保它是合法的可微标量）。

  整个函数即 $\texttt{exp\_z}(\theta)=\langle0|R_x(\theta)^\dagger Z R_x(\theta)|0\rangle=\cos\theta$。

```python
grad_z = K.grad(exp_z)
```
- 对 `exp_z` 做**自动微分**，得到一个新函数 `grad_z(θ)`，返回 $\frac{d\langle Z\rangle}{d\theta}=-\sin\theta$。无需手写差分。

```python
params = K.convert_to_tensor(np.arange(0, 2 * np.pi, 0.01))
```
- 生成一组 θ 采样点 $[0, 2\pi)$，步长 0.01，并转成后端张量。

```python
plt.plot(params, [exp_z(param) for param in params], label=r"$\langle Z\rangle$")
plt.plot(params, [grad_z(param) for param in params],
         label=r"$\frac{\partial \langle Z\rangle}{\partial \theta}$")
plt.legend()
plt.xlabel(r"$\theta$")
plt.show()
```
- 对每个 θ 同时算出 $\langle Z\rangle$ 和它的导数，画两条曲线：一条是 $\cos\theta$，一条是 $-\sin\theta$；加图例、横轴标签并显示。
